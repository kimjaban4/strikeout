# 마운드 심리전 게임 기획서

## 1. 게임 개요

### 제목
마운드 심리전

### 장르
웹 기반 야구 심리전 / 확률형 로그라이크 / 투수 시점 승부 게임

### 플랫폼
PC 웹 브라우저

### 핵심 콘셉트
플레이어는 투수가 되어 타자와 1대1 승부를 벌인다. 타자는 매 타석마다 숨겨진 노림수를 가지고 있으며, 플레이어는 투구 결과와 타자의 반응을 보고 노림수를 추리해야 한다. 단순히 좋은 공을 던지는 게임이 아니라, 타자가 원하는 공을 피하고 약점을 찌르는 심리전이 핵심이다.

### 한 줄 소개
상대 타자의 노림수를 읽고, 반대로 던져 3이닝을 막아내는 투수 심리전 야구 게임.

## 2. 기획 의도

기존 야구 게임은 타격, 수비, 주루 조작이 복잡하게 섞여 있다. 이 게임은 그중 투수와 타자의 승부만 분리해, 짧고 반복 가능한 심리전 게임으로 만든다.

플레이어가 매 투구마다 고민해야 하는 질문은 다음과 같다.

- 이 타자는 지금 어떤 구종을 노리고 있을까?
- 이전 공에 대한 반응은 힌트였을까?
- 스트라이크가 필요하지만, 정면 승부를 해도 될까?
- 볼을 던져 유인할까, 승부구를 던질까?
- 같은 구종을 반복하면 읽히지 않을까?

목표는 야구 규칙을 모두 구현하는 것이 아니라, 투수와 타자의 수 싸움에서 오는 긴장감을 만드는 것이다.

## 3. 핵심 재미

### 3.1 불완전 정보 추리
타자의 노림수는 처음에 공개되지 않는다. 플레이어는 타자의 스윙 여부, 타이밍, 파울, 헛스윙, 지켜보기 같은 반응을 보고 추리한다.

### 3.2 확률을 유리하게 만드는 선택
결과는 확률로 결정되지만, 플레이어의 선택이 확률을 바꾼다. 타자의 노림수와 다른 구종을 던지면 헛스윙이나 범타 확률이 올라가고, 노림수와 맞아떨어지면 장타 위험이 커진다.

### 3.3 로그라이크식 매 경기 변화
게임 시작마다 투수 능력치, 보유 구종, 상대 타자 능력치가 달라진다. 같은 전략이 항상 통하지 않게 만들어 반복 플레이 가치를 준다.

### 3.4 짧은 경기 목표
현재 기본 목표는 3이닝을 3실점 이하로 막는 것이다. 한 판이 짧고, 실패해도 다시 시도하기 쉽다.

## 4. 게임 목표

### 기본 클리어 조건
3이닝 동안 3실점 이하로 막으면 클리어.

### 패배 조건
4실점 이상 허용하면 경기 종료.

### 플레이 단위
한 경기 = 3이닝 수비.

## 5. 게임 진행 흐름

1. 새 경기 시작
2. 투수 능력치와 구종 랜덤 생성
3. 상대 타순 1~9번 랜덤 생성
4. 타자가 타석에 들어옴
5. 타자는 숨겨진 노림수를 가짐
6. 플레이어가 구종 선택
7. 투구 결과 계산
8. 타자 반응과 결과 표시
9. 플레이어가 노림수를 추리
10. 볼카운트, 아웃카운트, 주자, 실점 갱신
11. 타석 종료 시 다음 타자
12. 3아웃 시 다음 이닝
13. 3이닝 종료 또는 4실점 시 경기 종료

## 6. 플레이어 역할

플레이어는 투수다.

직접 조작하는 요소는 현재 구종 선택이다. 향후 코스 선택, 승부 성향, 특수 능력 등을 추가할 수 있다.

현재 플레이어가 판단해야 하는 요소:

- 어떤 구종을 던질지
- 타자의 노림수가 무엇인지
- 카운트 상황에서 승부할지 유인할지
- 체력 소모가 큰 구종을 쓸지 아낄지
- 같은 구종 반복 위험을 감수할지

## 7. 투수 시스템

### 7.1 투수 능력치

| 능력치 | 설명 |
|---|---|
| 구속 | 강속구 계열의 위력에 영향 |
| 제구 | 스트라이크존에 들어갈 확률과 볼넷 위험에 영향 |
| 변화 | 변화구/느린공 계열의 위력에 영향 |
| 멘탈 | 주자 상황 등 위기에서 능력 하락 방지 |
| 체력 | 투구를 반복할수록 능력 유지 |

### 7.2 구종

현재 구종 후보:

| 구종 | 분류 | 역할 |
|---|---|---|
| 포심 | 강속구 | 정면 승부용 |
| 투심 | 강속구 | 빗맞힘/땅볼 유도 |
| 커터 | 강속구 | 중심을 비껴 맞히게 유도 |
| 슬라이더 | 변화구 | 헛스윙 유도 |
| 커브 | 변화구 | 타이밍 교란 |
| 체인지업 | 느린공 | 직구 노림 저격 |
| 스플리터 | 느린공 | 낙차 큰 승부구 |

게임 시작 시 이 중 3~5개가 랜덤으로 주어진다.

### 7.3 체력

투구할 때마다 체력이 감소한다. 소모량은 구종마다 다르다. 체력이 낮아지면 구위와 제구가 떨어져 안타, 볼넷 위험이 커진다.

## 8. 타자 시스템

### 8.1 타자 능력치

| 능력치 | 설명 |
|---|---|
| 컨택 | 공을 맞히는 능력 |
| 파워 | 장타와 홈런 위험 |
| 선구 | 볼을 참는 능력 |
| 주력 | 안타/장타 결과에 영향 |
| 예측 | 노림수가 맞았을 때의 위력 |

### 8.2 타순별 성향

| 타순 | 성향 |
|---|---|
| 1~2번 | 출루형, 컨택/선구/주력 높음 |
| 3~4번 | 중심 타자, 파워와 예측 높음 |
| 5~6번 | 장타 변수, 파워는 높지만 약점 존재 |
| 7~9번 | 하위 타선, 전체 능력은 낮지만 예외적 장점 가능 |

## 9. 타자 노림수 시스템

타자는 매 타석마다 숨겨진 노림수를 가진다.

현재 노림수 종류:

- 강속구 노림
- 변화구 노림
- 느린공 노림

플레이어는 타자의 반응을 통해 노림수를 추정한다.

예시:

- 변화구를 노리는 타자에게 강속구를 던짐: 타이밍이 늦거나 지켜볼 가능성 증가
- 강속구를 노리는 타자에게 체인지업을 던짐: 너무 빠른 스윙, 헛스윙 가능성 증가
- 노림수와 같은 구종을 던짐: 완벽한 타이밍, 장타 위험 증가

## 10. 투구 결과 구조

공 하나의 결과는 아래 흐름으로 계산된다.

1. 공이 스트라이크존에 들어갔는지 결정
2. 타자가 스윙할지 결정
3. 스윙했다면 타이밍 계산
4. 컨택 여부 계산
5. 맞았다면 파울/범타/안타/2루타/홈런 계산
6. 볼카운트, 아웃카운트, 주자, 실점 반영

### 주요 결과

- 볼
- 루킹 스트라이크
- 헛스윙
- 파울
- 범타
- 안타
- 2루타
- 홈런
- 볼넷
- 삼진

## 11. UI 기획

현재 목표 UI는 어두운 야구 중계/게임캐스트 스타일이다.

### 11.1 화면 구성

| 위치 | 내용 |
|---|---|
| 왼쪽 | 투수 정보, 체력, 능력치, 구종 선택 |
| 상단 | 이닝, 실점, 목표, B/S/O, 주자 상황 |
| 중앙 | 투수-타자 승부 화면, 스트라이크존, 공 애니메이션 |
| 오른쪽 | 상대 타자 정보, 노림수 추정, 승부 기록 |

### 11.2 피드백 UI

투구 전에는 타이밍 피드백을 표시하지 않는다. 공이 도착한 후에만 짧게 표시한다.

표시 예시:

- 완벽
- 좋음
- 불안정
- 늦음
- 너무 빠름
- 완전히 속음

이 피드백은 노림수 추리의 핵심 힌트다.

## 13. 현재 수정 필요 또는 개선 후보

### 우선순위 높음

- UI 시각 검수와 정렬 안정화
- 구종 선택 버튼 가독성 개선
- 타자 반응 피드백을 더 직관적으로 표시
- 결과 확률을 화면에 일부 표시
- 타석 종료 후 자동 다음 타자 진행 옵션

### 우선순위 중간

- 코스 선택 추가: 몸쪽, 바깥쪽, 높은 코스, 낮은 코스
- 타자별 약점/강점 태그 추가
- 이닝 사이 로그라이크 보상 추가
- 특수 능력 추가
- 난이도 선택

### 우선순위 낮음

- 선수 이미지 또는 실루엣 개선
- 관중/구장 배경 고도화
- 사운드 효과
- 기록 저장
- 랭킹

## 14. 향후 확장안

### 14.1 코스 시스템

구종뿐 아니라 코스까지 선택하게 만든다.

예시 코스:

- 높은 코스
- 낮은 코스
- 몸쪽
- 바깥쪽

타자 노림수도 구종 + 코스 조합으로 확장할 수 있다.

### 14.2 로그라이크 보상

이닝 종료 후 보상 선택지를 제공한다.

예시:

- 제구 +5
- 체력 회복
- 특정 구종 강화
- 변화구 헛스윙률 증가
- 볼넷 위험 감소
- 다음 이닝 첫 타자 약점 공개

### 14.3 타자 성향 다양화

타자에게 고유 성향을 부여한다.

예시:

- 초구 공략형
- 2스트라이크 보호형
- 변화구 약점
- 강속구 킬러
- 볼넷 유도형
- 반복 구종 저격형

## 15. 밸런스 방향

이 게임은 완전한 정답 맞히기 게임이 되면 안 된다. 플레이어가 좋은 추리를 해도 100% 성공하면 긴장감이 사라진다. 반대로 운이 너무 강하면 선택의 의미가 사라진다.

권장 방향:

- 좋은 선택: 성공 확률이 크게 올라감
- 나쁜 선택: 장타/볼넷 위험이 커짐
- 그래도 야구답게 약간의 변수는 남김

목표 감각:

- 플레이어가 이긴 이유를 이해할 수 있어야 한다.
- 플레이어가 진 이유도 대체로 납득할 수 있어야 한다.
- 운이 개입하더라도 다음 선택을 바꾸고 싶어야 한다.
