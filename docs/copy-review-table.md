# 문구 전체 리라이트 검토표

상태: 검토용 초안. 아직 코드 반영 금지.

목표:
- AI식 설명문을 줄이고 실제 야구 현장/벤치/중계 톤으로 통일한다.
- 효과 수치는 유지하되, 플레이어에게 보이는 문장은 짧게 만든다.
- 선택지는 "작전판에 적힌 말"처럼 쓴다.

금지 표현:
| 금지 표현 | 교체 방향 |
|---|---|
| 반영됩니다 | 따라갑니다 / 이어집니다 / 바로 갑니다 |
| 선택지 | 카드 / 보상 / 후보 / 사인 |
| 흐름 | 배합 / 타이밍 / 사인 / 리듬 / 분위기 |
| 운영 | 리드 / 배합 / 작전 / 경기 운용 |
| 증가합니다 | 오른다 / 살아난다 / 열립니다 / 강해집니다 |
| 감소합니다 | 줄어든다 / 늦춘다 / 숨긴다 |
| 영향을 줍니다 | 흔듭니다 / 묶습니다 / 늦춥니다 |
| 확인해야 합니다 | 기억합니다 / 보고 갑니다 |
| ~할 수 있습니다 | ~합니다 / ~갑니다 / ~노립니다 |

## 7. 카드/태그 문구 규칙

카드 카탈로그는 양이 많아서 개별 문구를 전부 여기서 확정하기보다, 아래 규칙으로 전체 재작성하는 것이 좋습니다.

| 현재 패턴 | 수정 패턴 | 예시 |
|---|---|---|
| 제구가 증가합니다 | 제구 +N / 코너가 산다 | 포심 제구가 증가합니다 -> 포심 제구 +5 |
| 확률이 증가합니다 | 위험/헛스윙/땅볼 +N | 헛스윙 확률이 증가합니다 -> 헛스윙 +5% |
| 효과가 좋아집니다 | 더 먹힌다 / 더 늦춘다 / 더 숨긴다 | 다음 승부의 읽기가 좋아집니다 -> 다음 사인이 더 선명합니다 |
| 보상 선택지가 증가합니다 | 보상 +1장 | 이닝 미션 성공 시 보상 선택지가 증가합니다 -> 미션 성공 시 보상 +1장 |
| 흐름이 좋아집니다 | 카드 등급이 오른다 / 분위기를 가져온다 | 보상 흐름이 좋아집니다 -> 카드 등급이 오릅니다 |
| 운영 방식입니다 | 작전 카드입니다 | 카드는 단순 능력치가 아니라 운영 방식입니다 -> 카드는 오늘의 작전입니다 |

### 카드 문구 샘플

| 카드 | 현재 설명 | 수정안 |
|---|---|---|
| 포심으로 존 점유 | 포심으로 존 안을 밀어붙일 때 손끝이 더 안정됩니다. | 포심을 존 안에 꽂을 때 손끝이 안정됩니다. |

| 커브로 시선 끌기 | 커브를 보여주면 타자의 시선이 느린 궤적에 묶입니다. | 커브를 보여주면 시선이 느린 궤적에 묶입니다. |

| 포수의 첫 수 | 이닝 첫 타자를 상대로 포수가 안전한 흐름을 먼저 잡아줍니다. | 첫 타자부터 포수가 사인을 좁혀 줍니다. |
