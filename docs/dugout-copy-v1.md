# 덕아웃 이벤트 문구 1차 초안

상태: 검토용 초안. 아직 코드 반영 금지.

수정 기준:
- 덕아웃 이벤트는 타석 중 반응을 말하는 장면이 아닙니다.
- 덕아웃은 이닝 시작 전/이닝 사이에 벤치에서 작전, 자료, 수비 위치, 주심 존, 상대 타순을 정리하는 이벤트로 둡니다.
- `늦은 배트`, `앞발이 빠름`, `배트 중심 경고`, `몸쪽 반응` 같은 직전 투구 피드백은 포수가 그때그때 말해주는 실시간 콜로 분리합니다.
- 정답 보상과 실패 페널티는 차이가 확실해야 합니다.

## 분리 기준

| 구분 | 들어갈 내용 | 예시 |
|---|---|---|
| 덕아웃 이벤트 | 이닝 전 작전, 전력분석, 수비 위치, 주심 존, 상대 타순 대비 | 전력분석 메모, 감독 사인, 외야 깊게, 주심 낮은 존 |
| 포수 실시간 콜 | 직전 투구 반응, 배트 타이밍, 몸쪽 반응, 정타 경고 | 배트가 늦습니다, 앞발이 빠릅니다, 같은 공은 맞습니다 |

## 덕아웃 보상 강도

| 강도 | 성공 보상 | 실패 페널티 |
|---|---|---|
| 안전 | 보상 흡수 +1, 간파도 소폭 감소 | 간파도 소폭 증가 |
| 일반 | 보상 흡수 +2, 다음 승부 핵심 버프 1개 | 정타 위험 또는 간파도 뚜렷하게 증가 |
| 강함 | 보상 흡수 +3, 보상 카드 등급 상승에 크게 기여 | 장타/정타 위험 크게 증가 |
| 도박 | 보상 흡수 +4 또는 보상 +1장 | 실패 시 다음 타자/위험 타자 승부가 크게 불리 |
